# 08 - 非功能需求

## 1. 安全性
- 全链路 HTTPS 与敏感字段加密存储。
- 原始资源与外发资源物理/逻辑隔离。
- 下载令牌限时、限次、防盗链。
- 高风险操作启用二次认证。

## 2. 合规性
- 严格执行授权范围、期限、用途约束。
- 所有合同、审批、交付记录可追溯。
- 支持审计导出与监管抽查。
- 支持违规冻结与追责流程。

## 3. 可用性与性能
- 门户可用性目标：99.9%。
- 常规检索响应：P95 < 2s。
- 申请提交成功率：> 99.5%。
- 审批与计费接口支持幂等与重试。

## 4. 可维护性
- 规则、流程、模板配置化管理。
- 关键模块具备版本管理与回滚能力。
- 异常告警覆盖支付回调失败、交付失败、审计写入失败。

## 5. 数据管理
- 数据分级：公开预览、受控交付、内部原始数据。
- 周期备份与恢复演练。
- 交付文件生命周期管理（自动过期与清理）。

## 6. 可观测性
- 建立业务指标：申请量、通过率、审批时长、授权收入。
- 建立安全指标：下载异常率、越权尝试、违规事件数。
- 支持按申请号追踪端到端链路。
