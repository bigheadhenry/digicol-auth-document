# 02 - 标准与术语

## 1. 授权依据
平台规则依据以下制度建立：
- 《中华人民共和国民法典》
- 《中华人民共和国著作权法》及实施条例
- 《博物馆条例》《博物馆管理办法》
- 《文物复制拓印管理办法》
- 《博物馆馆藏资源著作权、商标权和品牌授权操作指引（试行）》
- 《数字资源管理办法》
- 《数字资源授权规定及费用表》

## 2. 术语
- **授权方**：数字资源权利主体或管理主体。
- **被授权方**：申请并获得授权使用数字资源的主体。
- **数字资源**：二维影像、视频、音频、三维模型、软件工程文件及衍生数据。
- **授权合同**：定义授权范围、期限、费用、交付、义务与违约责任的法律文件。
- **授权申请函**：申请用途、资源目录、资质信息与结算信息的正式材料。

## 3. 授权规则
### 3.1 授权模式
- 采用直接授权模式，由授权方与被授权方直接签约。
- 原则上为非专有许可。
- 若申请方、实际使用方、付款方不一致，需签三方合同。

### 3.2 授权范围
- 按合同限定使用主体、目的、方式、时间、地域、载体。
- 原则上不授予超范围再授权权利。

### 3.3 授权期限
- 标准期限：合同签署起 1 年内。
- 超期规则：超过 1 年需走升级审批；最长不超过 5 年。

### 3.4 使用限制
禁止以下利用情形：
- 危害国家统一、主权和领土完整。
- 违反公序良俗、职业道德。
- 涉及民族歧视、迷信邪教、色情低俗等违法违规内容。

### 3.5 使用义务
- 发布内容显著位置标注“故宫博物院提供”或等同表述。
- 出版物/实体物至少提交 2 份样本。
- 展览/线上展示需提交利用场景材料与使用情况报告。

## 4. 授权分类与审批口径
| 类别 | 典型场景 | 审批特征 | 备注 |
| :--- | :--- | :--- | :--- |
| 学术研究 | 论文、期刊、教学材料 | 简化审批 | 部分场景可免费（如20张以内） |
| 展览展示 | 墙面、展签、宣传品 | 常规审批 | 批量印制超阈值需议价 |
| 出版 | 一般出版物、拍卖图录 | 常规/升级审批 | 改版修订需重新授权 |
| 数字传播 | 网页/APP/电子书/节目 | 常规审批 | 按分辨率与用途计费 |
| 数字展览 | 固定主题的数字项目 | 升级审批 | 有比例、期限与成果共享约束 |
| 文创产业 | 商品开发、销售 | 升级审批 | 影像费 + 授权金 |
| 视频授权 | 素材/成片 | 升级审批 | 按分钟或按集计价 |
| 三维/数字展项 | 模型、程序安装包 | 升级审批 | 多按项目协商 |

## 5. 数据规格口径
### 5.1 二维影像
- 交付格式以 TIFF/JPG/PNG 为主，按用途配置。
- 数字展览常用交付基线：TIFF，300dpi。
- 费用单位通常按“张”计算。

### 5.2 视频
- 高清：MP4(H.264, 10Mbps)
- 4K：MP4(H.264, 20Mbps)
- 计费可按分钟（不足1分钟按1分钟）或按集。

### 5.3 数字展项
- 数据形态：软件工程文件安装包。
- 常见计价：按项目包干 + 按天服务费。

## 6. 价格与结算规则
### 6.1 计费结构
- 总费用 = 资源使用费 +（如适用）授权金 + 其他约定费用。
- 文创类一般采用“影像费用 + 授权金（按销售额比例）”。

### 6.2 计费要点
- 同一影像多用途使用可按折扣规则计算（2种/3种/4种以上）。
- 印量、分辨率、用途类型、期限是主要计费因子。
- 重印在授权期内可按规定免重新授权；改版/修订需重新授权。

### 6.3 支付与币种
- 支付条件：合同签署后按合同约定支付。
- 结算币种：人民币；港澳台及海外可支持美元。
- 手续费、转录等额外成本由申请方承担。

## 7. 申请材料标准
### 个人申请
- 用途说明、资源目录、身份证明、联系方式、开票信息。

### 单位申请
- 盖章申请函、用途说明、资源目录、资质证明、联系人信息。
- 申请方、合同签署方、付款方原则一致；不一致时补充第三方资质与协议。
- 明确付款方式与发票类型（普票/专票）及税务信息。
