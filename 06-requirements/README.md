# 06 - 功能需求清单

## 1. 外部申请门户
| 模块 | 功能 | 需求说明 | 优先级 |
| :--- | :--- | :--- | :--- |
| 账号与认证 | 主体注册 | 个人/单位双通道，单位需上传资质 | P0 |
| 资源检索 | 条件筛选 | 按资源类型、用途可授权性、规格筛选 | P0 |
| 申请中心 | 申请函在线填写 | 用途、期限、范围、资源目录、载体必填 | P0 |
| 材料管理 | 附件上传 | 支持资质、证明、第三方付款证明 | P0 |
| 报价确认 | 费用明细可视化 | 展示计费项、折扣、协商项 | P0 |
| 合同签署 | 电子签约 | 在线预览、签章、版本留存 | P1 |
| 支付管理 | 在线支付/线下回填 | 支持支付状态追踪与凭证上传 | P1 |
| 交付下载 | 下载令牌 | 限时、限次、防盗链，记录下载日志 | P0 |
| 履约反馈 | 样本与利用报告提交 | 支持图片/视频/文件上传与审核 | P1 |

## 2. 内部管理后台
| 模块 | 功能 | 需求说明 | 优先级 |
| :--- | :--- | :--- | :--- |
| 申请审核 | 材料校验与审批 | 通过/驳回/退回补正，保留审批意见 | P0 |
| 流程引擎 | 流程分级 | 支持按用途、金额、期限自动选流程 | P0 |
| 规则中心 | 定价配置 | 支持规则版本、启停、生效时间 | P0 |
| 合同中心 | 合同管理 | 模板管理、条款变量填充、补充协议 | P1 |
| 交付中心 | 交付编排 | 格式转换、水印、打包、下载链接生成 | P0 |
| 财务中心 | 收款与开票 | 回单核销、发票记录、异常对账 | P1 |
| 履约管理 | 样本与报告复核 | 标记是否履约，触发催交 | P1 |
| 风险控制 | 违规处置 | 黑名单、权限冻结、案件流转 | P1 |
| 报表中心 | 经营统计 | 收入、类型分布、审批时效、违规率 | P1 |

## 3. 关键业务规则需求
1. 申请方、签约方、付款方一致性校验。
2. 授权期限默认 1 年，超过阈值自动升级审批。
3. 超范围用途、改版修订、超印量触发重新授权。
4. 发布署名规则在交付时自动提示并在合同条款中固化。
5. 展览/线上展示项目必须提交利用报告后才可关闭履约。

## 4. 接口与集成需求
- 支付回调接口：幂等更新订单状态。
- 电子签章接口：签署、验签、存证。
- 身份认证接口：统一账号体系与单点登录。
- 对象存储接口：交付文件上传、下载控制、生命周期策略。

## 5. 验收标准
- 端到端流程可在系统内独立闭环完成。
- 关键规则可配置且规则变更可追踪。
- 审批、支付、交付、下载均可追溯到人和时间。
- 随机抽检 10 个授权项目可复原完整证据链。
